What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

98% Browser Compatibility
87KB Minified Size
15+ Years Active

Core Features

◈
DOM Traversal & Manipulation

Select elements with CSS-style selectors and modify content, attributes, and structure dynamically.

◈
Event Handling

Attach and manage events consistently across all browsers with simple methods like .on(), .click(), .hover().

◈
Effects & Animations

Built-in methods for show, hide, fade, slide, and custom animate with easing support.

◈
Ajax

Simplified asynchronous HTTP requests with $.ajax(), $.get(), and $.post() methods.

◈
Chaining

Apply multiple methods on the same selection in a single statement for concise code.
